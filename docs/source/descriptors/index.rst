Molecular Descriptors Reference
============================

Surfacia computes a comprehensive set of molecular descriptors based on quantum chemical properties. These descriptors provide physically meaningful insights into molecular structure, reactivity, and electronic properties.

Overview
--------

Surfacia's descriptors are designed to be **interpretable** and **physically grounded**. Unlike traditional molecular fingerprints, each descriptor corresponds to a specific molecular property that can be understood in chemical terms.

**Why This Matters**

- **Interpretability**: Each descriptor has clear physical meaning
- **Transferability**: Based on fundamental physics, not dataset-specific patterns
- **Intuition**: Chemists can understand what drives model predictions
- **Validation**: Can be compared with experimental observations

**Descriptor Categories**

1. **Size and Shape**: Geometric and structural properties (22 features)
2. **Electronic Properties**: Quantum mechanical electronic descriptors (28 features)
3. **MQSA Modes**: Multi-scale quantitative surface analysis (63 features)

**Total Features**: 113+ unique descriptors per molecule

How Descriptors are Calculated
---------------------------

All descriptors are computed from quantum mechanical wavefunctions generated by Gaussian and analyzed by Multiwfn:

1. **Wavefunction Generation**: Gaussian calculations (HF/DFT)
2. **Surface Analysis**: Multiwfn partitioning of vdW surface
3. **Feature Extraction**: Statistical analysis of surface properties
4. **Multi-scale Analysis**: Three complementary analysis modes

**Key Concepts**

- **Isosurface**: Molecular surface defined by constant electron density (0.01 a.u.)
- **Weighted Partitioning**: Surface points assigned to atoms by distance-weighting
- **Principal Axes**: Mass distribution axes for size/shape characterization
- **Local Properties**: Point-wise electronic properties (ESP, ALIE, LEAE)

Data Format
------------

Output files contain these standard columns:

- **Sample Name/ID**: Molecule identifier
- **SMILES**: Molecular structure representation
- **Target**: Experimental property (if provided)
- **[Descriptor Names]**: 113+ descriptor columns
- **Split Column**: Training/Test set designation

**Example CSV Structure**

.. code-block:: csv

   Sample,SMILES,Target,Atom_Number,Molecule_Weight,Isosurface_Area,Sphericity,Farthest_Distance,...
   molecule1,c1ccccc1,5.2,12,180.5,0.847,12.345,...
   molecule2,CC(=O)O,7.8,15,165.2,0.912,10.892,...
