Overview
--------

These tutorials provide hands-on experience with Surfacia's molecular surface analysis and machine learning capabilities. Each tutorial includes complete examples with sample data and expected outputs.

Tutorial Structure
------------------

Each tutorial follows a consistent structure:

1. **Objective**: What you'll learn and accomplish
2. **Prerequisites**: Required knowledge and setup
3. **Step-by-step Instructions**: Detailed walkthrough
4. **Expected Results**: What outputs to expect
5. **Troubleshooting**: Common issues and solutions
6. **Next Steps**: Related tutorials and advanced topics

Getting Started
---------------

To begin with the tutorials:

1. Ensure Surfacia is properly installed (see :doc:`../getting_started/installation`)
2. Download the tutorial data files
3. Follow the tutorials in order for best learning experience
4. Refer to the :doc:`../commands/index` for detailed command references

Sample Data
-----------

All tutorials use standardized sample datasets that demonstrate key concepts:

* **Basic molecules**: Simple organic compounds for learning fundamentals
* **Complex systems**: Larger molecules showcasing advanced features
* **Benchmark datasets**: Standard datasets for comparison with literature
* **Custom examples**: Specialized cases for specific applications
