Visualization API
=================

The visualization layer turns numerical outputs into plots, molecular views, and interpretation-friendly graphics.

What Belongs Here
-----------------

- SHAP summaries
- dependence plots
- structure-linked inspection views
- interactive visual outputs

Typical Responsibilities
------------------------

- present model interpretation clearly
- connect features back to molecules and regions
- support exploratory result review
- prepare figures for reporting and discussion

When to Use the Visualization API
---------------------------------

Use this layer when you want to:

- generate figures programmatically
- build custom result dashboards
- adapt Surfacia outputs for reports or presentations
- link model explanations to structural inspection
